Premium Hotel & Resort Landing Page Template

A modern, high-conversion landing page template designed specifically for hotels, boutique stays, and resorts. Built with a "mobile-first" approach using Tailwind CSS and vanilla JavaScript.

Features

Modern Design: Sophisticated typography and spacious layouts using Playfair Display and Plus Jakarta Sans.

Dynamic Room Gallery: Interactive filtering system for different room categories and a high-resolution modal gallery.

Fully Responsive: Optimized for desktops, tablets, and smartphones.

Lead Generation: Integrated reservation request form with success state feedback.

Easy Customization: Utility-first CSS (Tailwind) allows for rapid branding changes (colors, fonts, spacing).

Quick Start

Clone this repository.

Rename index.html if necessary (it is set as the entry point by default).

Open index.html in any browser to preview.

Customize the --accent variable in the <style> block to match your client's brand color.

Deployment

This template is ready for GitHub Pages. Simply upload the index.html file to a repository and enable "Pages" in the repository settings to get a live URL instantly.
